# Instrucciones de Uso

Dato: Este proyecto se complementa con Proyecto_Ejemplo_Front_End_1.

1. Clonar el proyecto.
2. Navegar a la carpeta del proyecto ```cd nombre-carpeta-proyecto```.
3. Reemplazar los valores de las variables de entorno del archivo 'environment.js'.
4. Ejecutar ```node server.js``` para iniciar un servidor local para manejar solicitudes HTTP.
